# Log Analyzer - Python

Mini ferramenta para analisar arquivos auth.log e identificar tentativas de acesso via SSH.

## Funcionalidades

- Extrai IPs das tentativas
- Conta número de tentativas por IP
- Identifica IP mais suspeito
- Gera relatório em arquivo

## Tecnologias

- Python 3

## Objetivo

Projeto de estudo focado em automação e fundamentos de Cybersecurity.
